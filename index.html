2 wheel moving robot (mini-mouse)


A project build to test my knowledge in PCB design using KiCad using an STM32 microcontroller, and C programming. The robot is able to move in a straight line, turn left and right, and avoid obstacles using ultrasonic sensors.

Home hosted server


Sensor integration for health monitoring of 3D-Printed structures (of new materials)

This project focused on developing a wireless, low-cost sensor integrated directly into 3D-printed structures for Structural Health Monitoring (SHM). Using Conductive PLA and Silver Nano Ink, the team designed and simulated humidity and strain sensors capable of detecting environmental changes through shifts in resonant frequency.

Software-Based Lock-In Amplifier

Software-Based LIA is an interface based on ULIA (Universal Lock-In Amplifier), a library that was developed and tested by Uhl et al. The purpose of this software is to process data received from a non linear experimental setup using the ULIA library, enabling real-time visualization of the processed data.

High-Precision Motion Tracking Module



This board is a 4-layer PCB designed for high-resolution motion tracking, navigation, and minimal vibration detection, based on ESP32-C3 (SoC) with an imbeded inverted-F antenna tuned at 50Ohm. this baord was devaloped to be part of a robot to detect the minimume vibration and to give a feedback about the oriantation and position of the robot using an IMU from boash (BMI330).

Circular Meander Antenna



Designed and simulated a compact meander-shaped antenna optimized to resonate at 730 MHz. Starting from a straight dipole model, the project explored rectangular and circular meander geometries to reduce antenna size while maintaining performance. Final simulations achieved a reflection coefficient (S11) of –12 dB and confirmed effective radiation in the XY plane.
